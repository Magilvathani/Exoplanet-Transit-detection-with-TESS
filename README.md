# Exoplanet-Transit-detection-with-TESS
This repository contains the code and documentation for my November astrophysics research project.  
The goal is to download light curves from the TESS mission and build a transit-search pipeline.

## Contents
- Data download using `lightkurve`
- Pre-processing (normalization, outlier removal)
- Transit detection using Box Least Squares (BLS)
- Plots & analysis
